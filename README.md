# NoKitty-Gamejam
**NoKitty** is a 2D Unity game created during the Golden Dolphin 21-Day GameJam. The project features a dialogue system, inventory system, trading system, save/load functionality, and a ScriptableObject-driven architecture. It demonstrates the implementation of core gameplay systems within a limited development timeline.



一款基于 **Unity** 开发的 2D 游戏作品，完成于 **金海豚奖 21 天限时游戏开发大赛（GameJam）**

本次作品围绕比赛主题 **「别按那个键」** 展开设计，在有限开发周期内独立完成了游戏核心功能与整体框架，包括：

-  剧情对话系统
- 背包与物品管理系统
- 商店交易系统
- 游戏存档系统
- ScriptableObject 数据驱动设计
- 多场景切换与关卡流程

该项目主要用于探索完整游戏开发流程，并积累 Unity 游戏架构设计与功能模块开发经验
